# Changelog

## v0.3.0

- Added `jobtrail followups` for stale active applications.
- Added top followups on the interactive home screen.
- Added application edit, archive, and unarchive commands.
- Added Excel, LaTeX, and all-format exports.
- Improved deterministic company and role extraction.
- Added local JSON backup export/import without secrets or tokens.
- Updated stats with active, archived, followups due, and manually verified counts.
- Expanded tests and documentation for daily workflow commands.

## v0.2.0

- Added interactive startup screen for `jobtrail`.
- Added onboarding wizard and settings menu.
- Added multiple provider accounts.
- Added provider-specific sync windows.
- Added Gmail date-window query support.
- Added Outlook provider stub for planned Microsoft Graph support.
- Added motivational phrase packs.
- Improved README and release docs.

## v0.1.0

- Added MVP local SQLite tracking.
- Added rule-based classification with confidence and reasons.
- Added Gmail provider abstraction.
- Added sample JSON sync.
- Added list, show, stats, and export commands.
- Added Gmail label dry-run/apply command.
- Added privacy-first defaults: no full email bodies by default and never delete email.
